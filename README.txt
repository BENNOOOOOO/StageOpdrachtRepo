    Clone Repository:

git bash op de locatie waar je wilt clonen en voer het volgende commando uit
git clone https://github.com/BENNOOOOOO/StageOpdrachtRepo.git


    Openen:

stop de zojuist gedownloade map in de root van je lokale server (bijvoorbeeld de htdocs map van MAMP)
Start de lokale server
Open een webbrowser en ga naar het adres van je lokale server en navigeer naar de bestandsmap

    Boekingen maken:

Vul het formulier in met de gebruiker en het gewenste tijdslot.
Klik op de "Boek" knop om een boeking toe te voegen.


    Overzicht van Boekingen en Gebruikers:

Onder "Alle Boekingen" zie je een lijst van alle gemaakte boekingen.
